<4강 웹문서의 이해>
-Web Developer를 통해 웹에서 여러가지 환경을 조작 가능
-웹은 웹+문서, 문서에 링크가 들어간 것 (웹 고안자: 팀 버너스 리)
-마크업랭귀지: 문서의 컨텐트를 그룹핑하거나 구분해주는 것
-HTML: 마크업랭귀지에 하이퍼텍스트를 단 것
-문서는 콘텐츠, 구조, 스타일이 있음.
-웹문서는 문서에 링크가 추가된 것
-콘텐트의 구성, 구분을 잘 해야할 것
	ㄴ제목 
	ㄴ문장 
	ㄴ목록 
	ㄴ표, 
	ㄴ폼

<12강 콘텐츠 목록 마크업>
-목록을 위한 태그는 ol, ul, dl이 있음
-ul >> 정렬되지 않은 항목을 갖고있는 목록
-ol >> 순서를 갖는 형태의 목록
-dl >> 정의 목록, 쌍을 이루어 목록을 만듦
